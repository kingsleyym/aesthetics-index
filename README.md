# Aesthetics Index

Medical aesthetics directory platform (MVP).

## Setup

```bash
pnpm install
pnpm dev
```

## Structure

- `apps/web` — Next.js web app
- `apps/widget` — Embeddable badge widget
- `packages/ui` — Shared UI components + tokens
- `packages/domain` — Domain entities + interfaces
- `packages/adapters` — Data providers (mock/firebase)
